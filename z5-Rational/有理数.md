# C++有理数类设计与运算符重载题目

## 题目要求

设计并实现一个Rational类，表示有理数（分数），要求完成以下功能：

1. **基本功能**：
   - 使用两个整数成员变量表示分子和分母
   - 构造函数（默认构造为0/1，带参数构造）
   - 约分功能（使用最大公约数算法）
   - 异常处理（分母为零的情况）

2. **运算符重载**：
   - 算术运算符：`+`, `-`, `*`, `/`
   - 自增运算符：前置`++`和后置`++`
   - 输入输出运算符：`>>`和`<<`
   - 类型转换运算符：转换为`double`、`float`和`int`

3. **其他要求**：
   - 实现一个简单的菜单驱动计算器界面
   - 所有运算结果应自动约分到最简形式
   - 代码规范，注释充分


## 扩展建议

1. 添加比较运算符（`==`, `!=`, `<`, `>`, `<=`, `>=`）
2. 实现复合赋值运算符（`+=`, `-=`, `*=`, `/=`）
3. 添加从浮点数构造有理数的功能
4. 实现解线性方程组的功能
5. 添加将假分数转换为带分数的功能
